Usage Guide

We hope you have a general idea of torrenting. Follow these steps to use our service:

  • Get the magnet link from any torrent directory you like (nyaa/TBP/rarbg, etc).
  • Paste the link in the placeholder and hit submit.
  • All the files in the torrent will appear as a list with checkboxes.
  • Select the files you want to download.
  • Hit the download button.
  • Please wait some time so the server can process the files (max 30s-1min).
  • You will get the file download notification.
